내부 스타일시트 : HTML문서 내부에 style 태그를 선언하여 해당하는 선택자에 스타일을 적용하고자 할 때 사용

외부 스타일시트 : HTML문서 외부에 별도의 스타일을 선언하는 문서를 구성하여 스타일을 적용하고자 할 때 사용
