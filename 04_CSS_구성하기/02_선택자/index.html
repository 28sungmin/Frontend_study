클래스 선택자 선언 시 주의사항

1. 클래스명을 선언 시 첫 글자는 숫자 불가 (단, 영문 이후에 포함되는 숫자는 상관없음)

2. 클래스명은 복수의 클래스명으로 선언이 가능 (복수 선언시, 반드시 한 칸을 띄우고 작성)

3. 클래스명 선언 시, 반드시 의미있는 이름으로 지어주는 것이 중요 (인수인계시 중요)

4. 클래스명으로 두 단어를 연결 시, 단어_단어 또는 wordWord

5. 클래스명은 재활용이 가능 (<->아이디 선택자)

아이디 선택자 선언 시 주의사항

1. 아이디명을 선언 시, 첫 글자는 숫자 불가 (단, 영문 이후에 포함되는 숫자는 상관없음)

2. 아이디명은 복수의 아이디명의 선언이 불가능

3. 아이디명 선언 시, 반드시 의미있는 이름으로 지어주는 것이 중요 (인수인계시 중요)

4. 아이디명으로 두 단어를 연결 시, 단어_단어 또는 wordWord

5. 동일한 아이디명은 한 HTML 문서에서 존재 불가 = 아이디명은 재활용 불가능 (두 명이 동일한 주민등록번호를 갖고 있는 경우) => (CSS 상에서는 스타일 적용이 되나, JS에서는 첫 번째 선언된 아이디명에만 동적 적용이 가능)

6. 아이디 선택자는 구조상 큰 규모를 대상으로 작성하는 것이 좋다. (명시도 부분에서 1개의 아이디당 100점)
